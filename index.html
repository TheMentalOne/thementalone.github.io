PROJECTS

Magic Inventory Control

Description


Magic Inventory Control is a decision support system that was designed to assist retail outlets to maximize profits by carefully monitoring customer preferences, product popularity and rate of purchase before restocking the inventory. Efficient inventory control reduces expenses hence, increases profits and customer loyalty.


University of Botswana Application

Description


The University of Botswana Online registration web portal was developed to computerize the manual process followed by prospective students when they apply to the institution. The system was designed to eliminate tedious manual processes, improve registration speed, access to info and decrese administration time.


BlackWareCrypt

Description


Despite being a JAVA application, BlackWareCrypt is one of my favourite application ideas. It is an encryption program that uses 4 different encryption ciphers to encrypt a piece of text. Decryption requires that the user understands the sequence to the key entry otherwise the text is re-encrypted with a new key.
